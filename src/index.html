ZEPHYR FORGE CLI

setup — Initialize and start your development environment.

init — Create a new Zephyr project.

dev — Start development for an existing project.

Available on: NPM Registry  |  GitHub Packages

Documentation: github.com/parazeeknova/zephyr

Note: These commands are designed to set up, initialize, or run development for your Zephyr environment.
